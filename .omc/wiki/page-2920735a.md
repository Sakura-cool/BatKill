# 图标生成

## 应用图标

图标通过 `generate_icon.sh` 脚本生成，输出到 `Resources/AppIcon.icns`。

### 生成流程
1. 脚本内嵌 Swift 代码，在 2048×2048 画布上绘制：
   - 绿色到蓝色渐变背景
   - 白色电池轮廓（线宽 56，圆角 100）
   - 白色电池端子（圆角 32）
   - 白色闪电符号（所有坐标乘 scale=2）
2. Swift 输出 2048×2048 PNG
3. `sips` 缩放到标准 icns 尺寸（16×16 到 1024×1024）
4. `iconutil` 打包为 `.icns`

### 缩放系数
所有绘制坐标乘以 `scale` 常量（当前 = 2），便于整体放大。修改 `scale` 即可统一缩放所有元素。
